## Getting Started

### How to Access the Admin Panel

1. **Log in as an Admin**
   - Go to PostVelocity website
   - Click "Login / Register" button
   - Enter your admin email and password
   - Make sure your account has admin permissions

2. **Open the Admin Panel**
   - After logging in, look at the top right corner of your screen
   - You will see a red button that says "👑 Admin Panel"
   - Click this button to open the admin dashboard
   - The admin panel will open as a large popup window

### Understanding the Admin Panel Layout

The admin panel has 4 main tabs at the top:
- **📊 Overview** - Shows key numbers and statistics
- **👥 Users** - Manage and view all users
- **📈 Analytics** - Detailed platform performance data
- **💳 Billing** - Revenue and payment information

You can click on any tab to switch between different sections.

## Overview Tab

The Overview tab gives you a quick snapshot of your entire platform.

### Key Metrics Cards

At the top, you'll see 4 important boxes:

1. **Total Users Box (Blue)**
   - Shows how many users are registered
   - Shows how many new users joined this week
   - Example: "25 total users, +3 this week"

2. **Monthly Revenue Box (Green)**
   - Shows how much money you made this month
   - Shows average money per user
   - Example: "$2,450 monthly revenue, $98/user average"

3. **Companies Box (Purple)**
   - Shows how many companies users have created
   - Shows new companies added this month
   - Example: "68 companies, +12 this month"

4. **Active Users Box (Orange)**
   - Shows users who logged in recently
   - Shows what percentage are active
   - Example: "18 active users, 72% active rate"

### Plan Distribution Section

This section shows how many users have each subscription plan:
- **Starter Plan** - Free or basic users
- **Professional Plan** - Mid-tier paying users
- **Business Plan** - Higher-tier paying users
- **Enterprise Plan** - Top-tier paying users

Each plan shows the number of users and how much money it brings in.

### Recent Transactions

At the bottom, you'll see the latest payments from users:
- Amount paid (like $29.99)
- What plan they bought
- Status (paid, failed, or pending)
- Date of the transaction

## Users Tab

The Users tab helps you manage all your platform users.

### User Actions Buttons

At the top of the Users tab, you'll find:

1. **+ Create Test User Button (Green)**
   - Click to add a new user for testing
   - Fill in name, email, and plan level
   - Useful for testing features

2. **🔄 Refresh Users Button (Blue)**
   - Click to reload the latest user information
   - Use this if you think data might be outdated

### Users Table

The table shows important information about each user:

- **User Column**: Shows the person's name and username
- **Email Column**: Shows their email address
- **Plan Column**: Shows their subscription level (with colored badges)
- **Posts Column**: How many social media posts they've created
- **Companies Column**: How many companies they've set up
- **Status Column**: If they're active and when they last logged in
- **Actions Column**: Buttons to view details or impersonate them

### How to Use the Actions

**View User Details:**
1. Find the user you want to learn more about
2. Click the "📊 Details" button in their row
3. A detailed popup will show their complete information

**Impersonate a User:**
1. Find the user you want to help or check on
2. Click the "👤 Impersonate" button in their row
3. You'll be logged in as that user to see what they see
4. To return to admin view, look for the "Stop Impersonating" button

## Analytics Tab

The Analytics tab shows detailed performance data for your platform.

### User Analytics Section

Shows information about your users:
- **Total Users**: How many people have signed up
- **Active Users**: How many people are using the platform
- **Admin Users**: How many administrators you have
- **Growth Rate**: Whether you're gaining or losing users (shown as percentage)

### Revenue Analytics Section

Shows information about money:
- **Monthly Revenue**: How much money you make per month
- **Annual Revenue**: How much money you make per year
- **ARPU** (Average Revenue Per User): How much each user pays on average

### Company Analytics Section

Shows information about companies created by users:
- **Total Companies**: How many companies exist on your platform
- **Average per User**: How many companies each user creates on average

### OAuth Analytics Section

Shows information about social media connections:
- **Total Connections**: How many social media accounts users have connected
- **Average per User**: How many social media accounts each user connects on average

## Billing Tab

The Billing tab helps you track money and payments.

### Revenue Overview Cards

Three important boxes at the top:

1. **Last 30 Days (Green)**
   - Shows money earned in the past month
   - Helps track recent performance

2. **Last 90 Days (Blue)**
   - Shows money earned in the past 3 months
   - Good for seeing longer trends

3. **Success Rate (Purple)**
   - Shows what percentage of payments succeed
   - Higher is better (you want close to 100%)

### Top Customers Table

Shows your best paying customers:
- **Customer**: Name and email of the user
- **Plan**: What subscription level they have
- **Total Spent**: How much money they've paid you overall
- **Transactions**: How many times they've made payments
- **Last Payment**: When they last paid you

### Plan Changes Section

Shows what users are doing with their subscriptions:

1. **Upgrades (Green)**
   - How many users moved to a higher plan this month
   - This is good - it means more money

2. **Downgrades (Yellow)**
   - How many users moved to a lower plan this month
   - Keep an eye on this - too many might be a problem

3. **Cancellations (Red)**
   - How many users cancelled their subscription this month
   - This is something to watch and try to reduce

## User Details Modal

When you click "📊 Details" on any user, a detailed popup opens.

### User Information Cards

At the top, you'll see 4 boxes with key information:

1. **Current Plan**: What subscription they have
2. **Total Spent**: How much money they've paid you
3. **Posts Generated**: How many social media posts they've created
4. **Health Score**: A number from 0-100 showing how engaged they are

### Companies Section

Shows all the companies this user has created:
- Company name
- What industry they're in
- When they created it

If they haven't created any companies, it will say "No companies created"

### Billing History Section

Shows all the payments this user has made:
- How much they paid
- What plan they bought
- If the payment succeeded or failed
- When the payment happened

If they've never paid, it will say "No billing history"

### How to Close the User Details

- Click the "×" button in the top right corner of the popup
- Or click anywhere outside the popup

## User Impersonation

Impersonation lets you log in as any user to help them or see what they're experiencing.

### How to Impersonate a User

1. Go to the **Users Tab**
2. Find the user you want to help
3. Click the **"👤 Impersonate"** button in their row
4. You will now see the platform exactly as they see it

### What You Can Do While Impersonating

- See their dashboard and all their data
- Create content for them
- Check their settings
- Experience any problems they might be having
- Test features from their perspective

### How to Stop Impersonating

1. Look at the top of the screen - you'll see a message saying you're impersonating someone
2. Click the **"Stop Impersonating"** button
3. You'll return to your normal admin view

**Important**: Always remember to stop impersonating when you're done helping a user!

## Tips and Best Practices

### Daily Admin Tasks

1. **Check the Overview Tab** - Look at key numbers each day
2. **Monitor New Users** - See who signed up recently
3. **Watch Revenue** - Make sure payments are processing correctly
4. **Check Failed Payments** - Help users with payment problems

### Weekly Admin Tasks

1. **Review User Activity** - See who's actively using the platform
2. **Check Plan Changes** - Monitor upgrades, downgrades, and cancellations
3. **Analyze Growth** - Look at user growth trends
4. **Review Top Customers** - Appreciate your best paying users

### Monthly Admin Tasks

1. **Deep Dive Analytics** - Study all the analytics data
2. **Revenue Analysis** - Compare this month to previous months
3. **User Health Review** - Check health scores and help struggling users
4. **Platform Performance** - Look for patterns and areas to improve

### Troubleshooting Common Issues

**If data looks wrong:**
1. Click the refresh button in any tab
2. Close and reopen the admin panel
3. Check if the user has the right permissions

**If you can't find a user:**
1. Click "🔄 Refresh Users" button
2. Make sure you're looking in the right place
3. Check if the user actually exists

**If payments look incorrect:**
1. Check the Billing tab for detailed information
2. Look at individual user billing history
3. Verify payment processor status

### Security Best Practices

1. **Always log out** when you're done using the admin panel
2. **Don't share admin credentials** with anyone who doesn't need them
3. **Use impersonation carefully** - only when helping users
4. **Regular password changes** for admin accounts
5. **Monitor admin activity** - keep track of who accesses what

### Getting Help

If you need help with the admin panel:
1. Check this guide first
2. Look for error messages and note them down
3. Take screenshots if something looks wrong
4. Contact technical support with specific details
